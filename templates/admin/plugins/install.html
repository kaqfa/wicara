Plugin Package Requirements

  • ZIP Format: Plugin must be packaged as a ZIP file
  • Directory Structure: ZIP should contain a single plugin directory
  • __init__.py: Plugin directory must contain __init__.py file
  • plugin.json: Manifest file with metadata (name, version, author, description)
  • BasePlugin: Plugin class must inherit from BasePlugin
Example Structure: 
my-plugin.zip
└── my-plugin/
    ├── __init__.py
    ├── plugin.py
    ├── plugin.json
    ├── templates/ (optional)
    └── static/ (optional)
