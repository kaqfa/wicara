About Groups

Site groups help you organize and manage multiple sites:

  • Hierarchical organization - Create nested group structures
  • Bulk operations - Manage multiple sites at once
  • Permission delegation - Assign roles at group level
  • Resource quotas - Set limits for entire groups
Tip: You can always move sites between groups later.
Group ID Rules
  • Must be 3-50 characters
  • Lowercase letters only
  • Numbers and hyphens allowed
  • Examples: client-a, eu-sites
